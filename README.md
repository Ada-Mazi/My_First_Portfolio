\## Features



\- Basic HTML pages with proper structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)  

\- Semantic HTML tags: `<header>`, `<main>`, `<footer>`, `<article>`, `<section>`, `<aside>`  

\- Navigation menus with links between pages  

\- Clickable images and embedded tweets  

\- Personal portfolio page showcasing projects, skills, and contact info  

\- W3C-compliant HTML
